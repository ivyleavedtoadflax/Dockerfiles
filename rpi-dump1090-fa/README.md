# rpi-dump1090-fa
This is a Alpine Linux based build of the FlightAware Dump1090 fork from https://github.com/flightaware/dump1090

Example:
```
docker run -t -i --device /dev/bus/usb:/dev/bus/usb bradsjm/rpi-dump1090-fa --help
```
